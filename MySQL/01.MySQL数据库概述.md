## 2.1 MySQL的安装

### 2.1.1 从二进制包安装

将MySQL的二进制软件包解压。

```bash
tar -zxvf mysql-xxx.tar.gz
```

将解压后的软件包移动到`/usr/local/`目录下，并将软件根目录重命名为mysql。
